# 智能选课系统

这是一个基于PyQt5开发的北京大学计算机方向智能选课系统，集成了DeepSeek AI智能评估功能。

## 🌟 功能特点

### 🎯 核心功能
- **个性化选课流程**：根据年级和专业进行个性化课程推荐
- **多类型课程支持**：必修课、选择性必修课、通识课程
- **AI智能评估系统**：基于DeepSeek AI的选课评估和建议
- **教师推荐**：为必修课程推荐优秀教师
- **实时学分统计**：动态显示已选学分和课程信息
- **精美界面设计**：现代化UI设计，支持动画效果

### 🤖 AI功能
- **DeepSeek AI集成**：使用先进的DeepSeek大语言模型
- **智能课程分析**：AI分析课程结构合理性
- **个性化建议**：基于专业和年级的定制化建议
- **学习规划**：短期、中期、长期学习规划建议
- **备用评估**：当AI不可用时自动切换到默认评估

### 📚 支持的专业
- 电子信息与技术
- 信息与计算科学  
- 智能科学与技术
- 应用物理学
- 计算机科学与技术
- 通班

### 🎓 支持的年级
- 大一到大四各年级

## 🚀 快速开始

### 环境要求
- Python 3.7+
- Windows 10/11 (已针对高DPI屏幕优化)
- 网络连接（用于AI功能）

### 一键启动
```bash
# 方式1：使用批处理文件（推荐）
双击 run.bat

# 方式2：使用Python脚本
python start_system.py

# 方式3：直接启动主程序
python main_enhanced.py
```

### 安装步骤

1. **克隆或下载项目**
   ```bash
   git clone [项目地址]
   cd CEP-5-26
   ```

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

3. **检查配置**
   ```bash
   python test_config.py
   ```

4. **测试AI服务**
   ```bash
   python check_ai_service.py
   ```

5. **启动系统**
   ```bash
   python start_system.py
   ```

## 📖 使用说明

### 选课流程

1. **欢迎界面** - 点击"开始选课"按钮
2. **年级选择** - 选择您的年级（大一到大四）
3. **专业选择** - 选择您的专业方向
4. **必修课选择** - 根据专业和年级自动加载相关必修课程
5. **选择性必修课** - 选择适合的选择性必修课程
6. **通识课选择** - 选择通识教育课程
7. **教师推荐** - 查看为您推荐的优秀教师
8. **AI智能评估** - DeepSeek AI分析您的选课并提供专业建议
9. **最终课表** - 查看完整的选课结果

## 🔍 故障排除

### 常见问题

**Q: 程序无法启动？**
A: 
1. 确保已安装Python 3.7+
2. 运行：`pip install -r requirements.txt`
3. 检查是否有杀毒软件阻止

**Q: AI评估不工作？**
A: 
1. 检查网络连接
2. 运行：`python check_ai_service.py`
3. 系统会自动使用默认评估

**Q: 界面显示异常？**
A: 
1. 确认图片资源文件存在
2. 检查屏幕DPI设置
3. 尝试重新安装PyQt5

**Q: 课程数据无法加载？**
A: 
1. 确认Excel文件路径正确
2. 检查文件格式是否正确
3. 运行：`python test_config.py`

### 系统检查工具

```bash
# 完整系统检查
python test_config.py

# AI服务专项检查
python check_ai_service.py

# 自动诊断启动
python start_system.py
```
